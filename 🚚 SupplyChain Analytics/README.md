# 🚚 SupplyChain Analytics

## Overview

SupplyChain Analytics is a CS50-inspired SQL portfolio project that simulates real-world logistics and transportation analysis.

The project focuses on delivery optimization, warehouse performance, revenue analysis, customer satisfaction, and operational efficiency using SQL.
