# CS50 SQL Project — Global Singers Database 🎵

This project is a structured SQL database built to analyze global music artists, their popularity, and career statistics.

It is designed as a learning project for strengthening **database thinking, data analysis, and real-world querying skills**.

---

# 📊 Database Overview

**Database:** `singers.db`
**Table:** `singers`

### Columns:
- `id` → unique singer ID
- `first_name`, `last_name` → artist name
- `country` → origin
- `genre` → music style
- `monthly_listeners` → streaming popularity
- `albums` → total albums
- `awards` → achievements
- `debut_year` → career start
- `retired` → retirement year

---

# 🧠 What You Will Learn

- How to filter large datasets using `WHERE`
- How to rank artists using `ORDER BY`
- How to calculate trends using `AVG()` and `MAX()`
- How to search text patterns using `LIKE`
- How to handle missing data (`NULL`)
