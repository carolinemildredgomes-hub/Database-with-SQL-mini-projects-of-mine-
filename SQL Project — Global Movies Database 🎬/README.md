# CS50 SQL Project — Global Movies Database 🎬

This project analyzes global movie data using SQL, including ratings, revenue, genres, and directors.

---

# 📊 Database Overview

**Database:** `movies.db`
**Table:** `movies`

### Columns:
- `id`
- `title`
- `director`
- `genre`
- `country`
- `release_year`
- `rating`
- `box_office`
- `duration`
- `lead_actor`

---

# 🧠 What You Will Learn

- Ranking using `ORDER BY`
- Filtering using `WHERE`
- Aggregation using `AVG`, `MAX`
- Pattern search using `LIKE`
- Time-based analysis using years
