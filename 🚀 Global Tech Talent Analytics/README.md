🚀 Global Tech Talent Analytics

📖 Overview

Global Tech Talent Analytics is an advanced SQL portfolio project designed to simulate real-world workforce analytics used by modern technology companies.

The project focuses on analyzing employee data, hiring trends, salaries, promotions, training effectiveness, and company performance using SQL.
