# The Missing Scholarship

SQL Database Investigation Project

## Overview

The Missing Scholarship is a database investigation project inspired by real-world university administration systems.

This project simulates the work of a data analyst responsible for investigating scholarship records, student financial transactions, and award distributions using SQL.

The goal is to strengthen practical SQL skills including:

* SELECT statements
* Filtering with WHERE
* INNER JOINs
* Subqueries
* Foreign keys
* Aggregation
* Database investigation techniques
* Data auditing and verification

## Project Objectives

Investigate three separate university mysteries:

1. The Missing Scholarship
2. The Wrong Recipient
3. The Forgotten Grant

For each case, SQL queries must be written to discover:

* Student information
* Scholarship recipients
* Financial records
* Payment history
* Data inconsistencies

## Technologies Used

* SQLite3
* SQL
* Relational Database Design

## Learning Outcomes

After completing this project, students will understand:

* Relational database structures
* Entity relationships
* Multi-table joins
* Nested queries
* Data tracing techniques
* Real-world database auditing workflows

Modern organizations rely heavily on relational databases. Data analysts, software engineers, database administrators, and backend developers frequently investigate data inconsistencies similar to those presented in this project.
