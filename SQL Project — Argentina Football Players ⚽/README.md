# CS50 SQL Project — Argentina Football Players ⚽

This project analyzes professional football players from Argentina using SQL queries. It focuses on player performance, clubs, goals, assists, and career data.

---

# 📊 Database Overview

**Database:** `argentina_players.db`
**Table:** `players`

### Columns:
- `id` → player ID
- `first_name` → first name
- `last_name` → last name
- `club` → current club
- `position` → playing position
- `goals` → total goals
- `assists` → total assists
- `matches` → total matches played
- `country` → nationality
- `market_value` → transfer value
- `debut_year` → career start year

---

# 🧠 Skills Practiced

- Ranking players by performance
- Filtering clubs and positions
- Finding top scorers
- Handling missing values
- Aggregating stats using SQL

---

# 🔍 Example Queries

- Find Lionel Messi’s club
- Top 10 goal scorers
- Average market value of defenders
- Players debuting after 2020
